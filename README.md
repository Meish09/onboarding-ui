# Onboarding UI 
Simple User Registration form.

## Languages Used:
React js, Html, css, javascript

## Setup

Make sure to install the dependencies

```bash
npm install
```

## Development

Start the development server on http://localhost:3000

```bash
npm start
```

## Preview

Deployed the application on Heroku:
